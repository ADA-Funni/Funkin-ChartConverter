# ADA_Funni's Funkin' Chart Converter
Friday Night Funkin' chart converter (Written as a command line program)

As of right now, this only converts from psych/legacy to vslice.

## How to use:
Throw Psych charts and songs into `assets/songs` and `assets/data`, just like when modding.
Then run `haxe build.hxml`, and they'll appear in the `export` folder in the V-Slice format!
